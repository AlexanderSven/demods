Руководство по использованию AquariusDS
===========================================
Данная система предназначена для проектирования иерархии нечётких автоматических классификаторов, а также для непосредственно классификации согласно выстроенной в системе иерархии.

Порядок оформления программного окружения
--------------------------------------------
Для запуска системы требуется установить на компьютер **Microsoft Visual Studio 2017**. Редакция **Microsoft Visual Studio 2017 Community Edition** является абсолютно бесплатной и её можно загрузить, пройдя по [данной ссылке](https://www.visualstudio.com/thank-you-downloading-visual-studio/?sku=Community&rel=15). После загрузки нужно запустить загруженный .exe-файл и установить в любую удобную для Вас директорию. Кроме **Microsoft Visual Studio 2017** никакого дополнительного ПО или библиотек для запуска не требуется.

Порядок установки Aquarius на компьютер
--------------------------------------------
Чтобы установить Aquarius, необходимо загрузить содержимое репозитория demods с сайта GitHub.com в виде ZIP-архива, либо же через систему Git, и распаковать в любом удобном для Вас месте.

Порядок запуска
--------------------------------------------

###Запуск решения Visual Studio###

Для запуска системы Aquarius с пользовательским интерфейсом в Visual Studio, необходимо: 
1. Открыть папку, куда данный репозиторий был распакован.
2. Из 3-х папок открыть папку Aquarius.
3. В папке Aquarius открыть файл Aquarius.sln с помощью Microsoft Visual Studio 2017.
4. В появившемся окне Visual Studio нажать кнопку F5.
5. Использовать программу для собственных нужд:)

###Запуск релизной сборки приложения###

Для запуска собранного автономного приложения, необходимо:
1. Открыть папку, куда данный репозиторий был распакован.
2. Открыть папку "Aquarius vX.X", где X.X - номер версии.
3. Если на компьютере отсутствует Visual Studio, рекомендуется обновить распространяемые библиотеки Visual C++ 2015. Установочные файлы данных библиотек можно найти в папке "Aquarius vX.X/Resources/Redistributable/".
4. В зависимости от архитектуры операционной системы, выбрать папку "х86" или "х64" соответственно.
5. Запустить Aquarius.exe
6. Использовать программу и наслаждаться жизнью:)
